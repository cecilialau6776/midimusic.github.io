EplayOrgan

Author: David Back, November, December 2017, March, June 2018

EplayOrgan, Multiple Organ Simulator

Version 2.2a Now Available for download

EplayOrgan is a brand new multi purpose organ intended to be playable by organists and versatile enough to be useful to everybody with an interest in organs. It will load and be playable almost instantly and does not need an expensive computer with huge amounts of memory.

This organ was designed and tested on a 64 bit Windows 10 system but was always intended to be versatile in that it will run on any Windows system back to and including Windows XP. It will also run on any Mac or Linux system under Wine. Latency is negligable under Windows.

I am still gaining experience with it myself and invite comments from others, particularly experienced organists, on its merits or problems with playability. My contact details are in About on this website. I will endeavour to cure any problems which arise and issue new versions as appropriate.

  

Main Features

  1. Easy and quick to install.
  2. All included organs are fully set up, just connect one or more midi keyboards, select ports, select your organ and begin playing.
  3. The correct midi channels are automatically set up for each organ.
  4. The keys, stops and pistons are all fully animated and their state is easily visible.
  5. Coupled keys move to show their state.
  6. See the Wurlitzer Band Organ keys and stops moving for the first time ever.
  7. Optional individual sound font for each division.
  8. Up to four stereo audio outputs with unlimited polyphony.
  9. All organs have the same basic layout and are user configurable.
  10. Organs included: Miditzer 160, 216 and 260 cinema organs

  11. More organs can be easily added.
  12. Built in midi player and recorder.
  13. Version 2.1 adds a built in playlist to each organ. This provides continuous playing of multiple midi files.
  14. Version 2.1a now includes Bruce Miles stereophonic organ soundfont, updated for eplayOrgan.
  15. Version 2.1b has minor improvements to eplayOrgan menus and soundfont.
  16. Version 2.1c bug fix version. Topmost C now displays correctly and "Cannot load pedal font" problem cleared + later bug.
  17. Version 2.1d stops may now be "swiped" open or closed, Manuals have a fully chromatic "swipe". Many other detail improvements.
  18. Version 2.2 the stops are coloured and re-arranged into standard positions for all organs. Other detail improvements.
  19. Version 2.2a the stops have 3 automatically selected font sizes, giving a better full screen display. Many detail improvements.
