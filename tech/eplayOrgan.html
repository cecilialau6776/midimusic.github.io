EplayOrgan

Author: David Back, November 2017 to March 2019

EplayOrgan, Multiple Pipe Organ Simulator and Translator

Version 3.0d Now Available for download

EplayOrgan is a multi purpose sample based pipe organ simulator intended to be playable by organists and versatile enough to be useful to everybody with an interest in organs. It will translate, play and record organ midi files for most organs in real time. It will load and be playable almost instantly and does not need an expensive computer with huge amounts of memory.

This organ was designed and tested on a 64 bit Windows 10 system but was always intended to be versatile in that it will run on any Windows system back to and including Windows XP. It will also run on any Mac or Linux system under Wine. Latency is negligable under Windows.

Main Features

  1. Easy and quick to install.
  2. All included organs are fully set up, just connect one or more midi keyboards, select your organ and begin playing.
  3. The correct midi channels are automatically set up for each organ.
  4. The keys, stops, pistons and two swell indicators are all fully animated and their state is easily visible.
  5. Hardware stop and expression controls may be input in Universal NRPN format. The same NRPN's will work for all organs.
  6. Optional Second Touch is available on all organs. See the second touch keys move to show exactly what is happening
  7. Optionally velocity sensitive on every rank. Useful for piano ranks.
  8. All stops are coloured and may be "swiped" open or closed, Manuals have a fully chromatic "swipe".
  9. The stops have automatically selected font sizes, varying with display size which makes them easy to read.
  10. Coupled keys move to show their state.
  11. See the Wurlitzer Band Organ keys and stops moving for the first time ever.
  12. Organs included: Miditzer 160, 216 and 260 cinema organs
    Wurlitzer 125, 150 and 165 band organs
    Hauptwerk: St Anne's Mosely, Burea Church, Paramount 310, Armley Schulze Leeds, Willis Salisbury, Cavaille-Coll Caen, Eisenbarth Friesach
    Estey Church Organ
    Viscount Regent and a Viscount Church organ for use with eplayWin32
    Allen Generic Church Organ and a Universal Organ which is used for translating organ midi files.
  13. More organs can be easily added.
  14. Outputs from any organ can be translated to Universal Organ format and optionally recorded.
  15. Inputs or recordings in Universal Organ format can be translated to any other organ format and optionally recorded.
  16. All organs have the same basic layout and are user configurable. New organs can be imported and existing organs exported.
  17. Optional individual .sf2 sound font for each division. Many useful sound fonts are included.
  18. Includes Bruce Miles stereo organ soundfonts, updated for eplayOrgan, and Rich Olsen's Band Organ fonts.
  19. Up to four stereo audio outputs with adjustable reverb and unlimited polyphony.
  20. Built in midi player and recorder using the latest technology.
  21. Multiple playlists. Provides continuous playing of multiple midi files. No other organ has this feature.
  22. Built in two tone metronome.
